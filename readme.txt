== Description ==

Wordpress Login Styles is a login style plugin. This plugin will adds an awesome login screen in your wordpress login. You can select login theme style & others color customization by using the options of this plugin.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `wp-login-styles` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Now go to Wp Login optins menu `Dashboard>> under Settings menu 
4. Now choose your settings click into save and logout and see the changes.....
